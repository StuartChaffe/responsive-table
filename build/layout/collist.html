Collist

Collist is a responsive layout system that allows you to arrange a list of elements in a grid-like fashion. To use it, simply add the collist class to a div. Like col, class row-n (where n is 1-12) provides a base width. You can then follow this with classes that specify how many items to display per row at particular breakpoints. For example, the class md-row-3 will show 3 items per row for viewport widths greater than the md breakpoint. Larger breakpoints take precedence over smaller ones.
