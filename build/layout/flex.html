Flex

Flex is a layout system that utilises flexbox to provide an easy to way to quickly markup simple layouts. Below are some simple examples of what's possible using flex. Make sure to check for flexbox browser support before using this in your project. Jazcss provides is very basic fallback for browsers that don't support flexbox. Check the layout/flex.scss file to see how it works.

As for learning how to use flexbox, here are some handy resources you should find useful:

  • CSS-Tricks guide and cheatsheet
  • Frogger-like game that teaches flexbox
  • Tower defense version
