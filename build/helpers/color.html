Color

A project's brand colours should be added to the $colors map in _variables.scss. If you don't have any particular names in mind for each colour, this site has suitable names based on hex colours. color-blindness.com also provide a similar tool which also shows the hue for specific colours.

Qubase provides a way to apply your $colors colours as background-colour or text-colour using the .bg-colourname and .fg-colourname classes like so:

<div class="padding-md bg-red fg-white">

<div class="padding-md bg-green fg-nero">

Color mixin

To use your $colors variables in your actual SCSS, you can use the color mixin like so:

background-color: color(red);
