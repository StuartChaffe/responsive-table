Carousel

This carousel uses bxslider behind the scenes.

You can use it with the handlebars-layout extension syntax:

{{#extend "carousel"}}
    {{#content "carousel-content"}}
        <li><img src="/images/banner-sample.jpg" /></li>
        <li><img src="/images/banner-sample.jpg" /></li>
        <li><img src="/images/banner-sample.jpg" /></li>
        <li><img src="/images/banner-sample.jpg" /></li>
    {{/content}}
{{/extend}}

Or alternatively just use the carousel code directly:

<div class="carousel">
    <div class="carousel__control carousel__control-left">
        <i class="fa fa-chevron-left" aria-hidden="true"></i>
    </div>
    <ul class="carousel__bxslider">
        <li><img src="/images/banner-sample.jpg" /></li>
        <li><img src="/images/banner-sample.jpg" /></li>
        <li><img src="/images/banner-sample.jpg" /></li>
        <li><img src="/images/banner-sample.jpg" /></li>
    </ul>
    <div class="carousel__control carousel__control-right">
        <i class="fa fa-chevron-right" aria-hidden="true"></i>
    </div>
</div>
